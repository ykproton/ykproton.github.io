Spring Boot Actuator：生产就绪功能

Spring Boot 包含许多附加功能，应用程序发布到生产环境后,可以帮助监控和管理应用。可以选择使用 HTTP endpoints 或 JMX 来管理和监控应用程序。审计、健康和指标收集也可以自动应用到应用上。
